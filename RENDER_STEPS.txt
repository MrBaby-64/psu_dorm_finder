TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

Problem: Routes not showing
Solution:
  - Check Render logs for errors
  - Verify API key is correct (no extra spaces)
  - Try clearing browser cache (Ctrl+F5)

Problem: Still redirects to Google Maps
Solution:
  - Deployment might not be complete
  - Hard refresh browser (Ctrl+Shift+R)
  - Check Render says "Deploy live"

Problem: "Failed to calculate route" or opens Google Maps
Solution:
  - This means API limit exceeded (500/day)
  - System automatically opens Google Maps for proper directions
  - This is the intended fallback behavior
  - Check API usage at https://www.graphhopper.com/
  - Limit resets daily at midnight UTC
